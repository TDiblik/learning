# Leetcode

This is a repository for my personal use to track my progress as I work through various coding challenges. 

It primarily contains solutions to LeetCode problems that I find interesting. 
To structure my learning, I am generally following the progression outlined in the [NeetCode Roadmap 250](https://neetcode.io/roadmap).

## How to Run
Compile and execute solutions directly via the included Makefile: `make [dev|prod] <problem-name> <solution-name>`.

For example: `make dev 1-Two-Sum final`
