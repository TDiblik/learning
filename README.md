# Learning

This repository serves as my personal knowledge base and workspace for various learning endeavors. It is structured into directories based on topics, books, or specific platforms.

## `leetcode/`

This directory is for tracking my progress as I work through various coding challenges, primarily solutions to LeetCode problems.
To structure my learning, I am generally following the progression outlined in the [NeetCode Roadmap 250](https://neetcode.io/roadmap).

### How to Run Leetcode Solutions
Compile and execute solutions directly via the included Makefile: `make <problem-name> <solution-name>`.

For example: `make 1-Two-Sum final`
